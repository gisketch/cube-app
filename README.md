# Cube

Vite + React + TypeScript starter wired with:

- Tailwind CSS + a small shadcn/ui setup
- React Router
- Dark mode (class-based) toggle
- Absolute imports via `@/*`
- ESLint + Prettier

## Scripts

```powershell
npm install
npm run dev
```

```powershell
npm run build
npm run preview
```

```powershell
npm run lint
npm run format
npm run format:check
```
